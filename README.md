# TrayTayo Web System

A modern Filipino food delivery web system.

## Quick Start

1. **Clone or download** this repo.
2. Open the main folder in your IDE or local server.
3. Open `index.html` in your browser to view the site.

## Features
- Responsive, mobile-friendly design
- Accessibility (WCAG AA, keyboard, ARIA)
- Optimized images (WebP, lazy-loading)
- Fast font loading, deferred scripts
- Safe-area insets for iOS
- Reliable cart and forms (validation, error states)
- Modern UI/UX with consistent components

---
For development: edit HTML/CSS/JS files directly. No build step required.
